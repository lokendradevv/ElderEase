# ElderEase 🧓❤️

ElderEase is a smart healthcare and safety companion application designed specifically for elderly individuals and their caregivers. The platform provides real-time health monitoring, medication management, emergency assistance, location tracking, and communication features to improve the quality of life and independence of senior citizens.

## 🌟 Features

### 👵 Elder Dashboard

* Personalized home screen
* Health status monitoring
* Daily activity tracking
* Quick access to emergency features
* User profile management

### 💊 Medication Management

* Add and manage medications
* Medication schedules and reminders
* Dosage tracking
* Medication history records

### 📍 Real-Time Location Tracking

* Secure GPS location sharing
* Caregiver access to elderly location
* Emergency location transmission
* Safe zone monitoring

### 🚨 Emergency Alert System

* One-tap emergency SOS
* Instant caregiver notifications
* Emergency contact management
* Real-time alert delivery

### 📷 Smart Monitoring

* Camera integration
* Safety monitoring features
* Activity verification support

### 👨‍👩‍👧 Caregiver Dashboard

* Monitor elderly family members
* Receive emergency alerts
* View health information
* Access activity and medication records
* Track location when authorized

### 🌐 Multi-Language Support

* Localization support
* Easy-to-use interface
* Senior-friendly design

## 🚀 Installation

### Prerequisites

* Flutter SDK
* Dart SDK
* Firebase Project
* Android Studio / VS Code
* Xcode (for iOS)

### Clone Repository

```bash
git clone https://github.com/lokendradevv/ElderEase.git
cd ElderEase
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```

### Build Web Version

```bash
flutter build web --release
```

## 🔥 Firebase Configuration

1. Create a Firebase Project.
2. Enable Authentication.
3. Create a Firestore Database.
4. Enable Cloud Messaging.
5. Download:

   * `google-services.json` (Android)
   * `GoogleService-Info.plist` (iOS)
6. Place configuration files in their respective directories.
